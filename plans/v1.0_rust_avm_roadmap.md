# Nexa v1.0 Rust AVM + WASM 实现路线图

## 概述

从 Python 脚本解释转译模式跨越至基于 Rust 编写的独立编译型 Agent Virtual Machine (AVM)，并引入 WASM 安全沙盒。

## 实现阶段

### Phase 1: 编译器前端 (Lexer + Parser)

**目标**: 用 Rust 实现完整的 Nexa 语法解析器

**步骤**:
1. 创建 `avm/` 目录结构
2. 实现词法分析器 (使用 `logos` crate)
3. 实现语法解析器 (使用 `lalrpop` 或手写递归下降)
4. 定义 AST 类型
5. 编写测试确保与 Python 解析器输出一致

**依赖**:
```toml
[dependencies]
logos = "0.14"
lalrpop = "0.20"  # 可选
thiserror = "1.0"
```

**输出**: `.nx` -> AST (Rust 结构体)

---

### Phase 2: 字节码编译器

**目标**: 将 AST 编译为 Nexa 字节码

**步骤**:
1. 设计字节码指令集
2. 实现编译器 (AST -> Bytecode)
3. 实现反汇编器 (调试用)
4. 设计 `.nxc` 编译产物格式

**核心指令**:
```rust
pub enum Opcode {
    // Agent 操作
    AgentCreate,
    AgentRun,
    AgentClone,
    
    // DAG 操作
    Pipeline,
    Fork,
    Merge,
    Branch,
    
    // 记忆操作
    MemStore,
    MemLoad,
    
    // 工具调用
    ToolCall,
    
    // 控制流
    Jump,
    JumpIf,
    
    // LLM
    LLMCall,
}
```

**输出**: AST -> `.nxc` 字节码文件

---

### Phase 3: AVM 运行时 (VM)

**目标**: 实现字节码解释器

**步骤**:
1. 实现栈式虚拟机
2. 实现异步调度器 (基于 Tokio)
3. 实现 Agent 运行时
4. 实现 LLM 集成 (HTTP 客户端)
5. 实现记忆管理

**依赖**:
```toml
[dependencies]
tokio = { version = "1", features = ["full"] }
reqwest = { version = "0.12", features = ["json"] }
serde = { version = "1", features = ["derive"] }
serde_json = "1"
```

**输出**: `avm run program.nxc`

---

### Phase 4: WASM 沙盒

**目标**: 实现 WASM 工具执行沙盒

**步骤**:
1. 集成 WASM 运行时 (wasmtime 或 wasmi)
2. 实现 WASI 接口
3. 实现资源限制
4. 实现 host functions
5. 创建 WASM SDK

**依赖**:
```toml
[dependencies]
wasmtime = "20"  # 或 wasmi
```

**输出**: 安全的工具执行环境

---

### Phase 5: FFI 与集成

**目标**: 提供 Python 和其他语言的 FFI

**步骤**:
1. 使用 PyO3 实现 Python 绑定
2. 实现 C FFI
3. 编写集成测试
4. 性能优化

**依赖**:
```toml
[dependencies]
pyo3 = { version = "0.21", features = ["extension-module"] }

[lib]
crate-type = ["cdylib", "rlib"]
```

## 技术决策

### 解析器选择
- **选项 A**: `lalrpop` (LR 解析器生成器)
- **选项 B**: 手写递归下降解析器
- **推荐**: 手写递归下降，更灵活，更好的错误信息

### WASM 运行时选择
- **选项 A**: `wasmtime` (快速，功能完整)
- **选项 B**: `wasmi` (纯 Rust，更易嵌入)
- **推荐**: `wasmtime` 用于生产，`wasmi` 用于测试

### 异步运行时
- **选择**: `tokio` (Rust 异步标准)
