# Nexa v1.3.2 Release Notes

**发布日期**: 2026-04-23  
**主题**: Error Propagation (? + otherwise)

## 概述

Nexa v1.3.2 引入了 Rust 风格的错误传播操作符，让 Agent 编程中的错误处理更加简洁和安全。

## 新特性

### P1-2: Error Propagation — 82 tests

**三个核心操作符**：

| 操作符 | 语法 | 语义 |
|--------|------|------|
| `try` | `try { ... }` | 尝试操作，捕获错误 |
| `?` | `expr ?` | 传播错误向上（类似 Rust 的 `?`） |
| `otherwise` | `expr otherwise fallback` | 错误时提供回退值 |

**语法示例**：

```nexa
// ? 操作符 — 向上传播错误
let value = parse(input) ?

// otherwise — 提供回退值
let result = risky_operation() otherwise "fallback"

// try — 尝试执行并捕获错误
try {
    data = fetch(url)
    result = process(data) ?
} catch (e) {
    result = handle_error(e)
}
```

**新增文件**：

| 文件 | 说明 |
|------|------|
| `src/runtime/result_types.py` | NexaResult, NexaOption, ErrorPropagation, propagate_or_else, try_propagate, wrap_agent_result |

**Agent-Native**: `wrap_agent_result()` 将 Agent 返回值自动包装为 NexaResult，便于 `?` 传播。
