# Nexa v1.3.3 Release Notes

**发布日期**: 2026-04-23  
**主题**: Background Job System

## 概述

Nexa v1.3.3 引入了后台任务处理 DSL，支持优先级队列、Cron 调度和重试逻辑。

## 新特性

### P1-3: Background Job System — 73 tests

**Job DSL 语法**：

```nexa
job SendEmail on "emails" (retry: 2, timeout: 120) {
    perform(user_id) {
        // 执行任务逻辑
    }
    on_failure(error, attempt) {
        // 失败回调
    }
}
```

**关键组件**：

| 组件 | 说明 |
|------|------|
| `JobSpec` | 任务规格定义 |
| `JobPriority` | 优先级 (Low/Normal/High/Critical) |
| `JobStatus` | 状态 (Pending/Running/Completed/Failed/Cancelled) |
| `BackoffStrategy` | 退避策略 (Fixed/Exponential/Linear) |
| `JobRegistry` | 任务注册表 |
| `JobQueue` | 优先级队列 |
| `JobWorker` | 工作线程 |
| `JobScheduler` | Cron 调度器 |

**新增文件**：`src/runtime/jobs.py`
