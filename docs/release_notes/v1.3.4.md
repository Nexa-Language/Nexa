# Nexa v1.3.4 Release Notes

**发布日期**: 2026-04-23  
**主题**: Built-In HTTP Server

## 概述

Nexa v1.3.4 引入了声明式 HTTP Server DSL，支持 CORS、CSP、静态文件、中间件和路由定义。

## 新特性

### P1-4: Built-In HTTP Server — 94 tests

**Server DSL 语法**：

```nexa
server 8080 {
    static "/assets" from "./public"
    cors { origins: ["*"], methods: ["GET", "POST"] }
    route GET "/chat" => ChatBot
    route POST "/analyze" => DataExtractor |>> Analyzer
}
```

**关键特性**：

| 特性 | 说明 |
|------|------|
| CORS 配置 | `CorsConfig` — origins, methods, headers, credentials |
| CSP 配置 | `CspConfig` — 安全内容策略 |
| Security Headers | `SecurityConfig` — 自动安全头注入 |
| Static Files | 目录映射和 MIME 类型处理 |
| Route Guards | ContractViolation 集成路由守卫 |
| Semantic Routing | 语义路由 (自然语言条件) |
| Hot Reload | 开发模式热重载 |
| Multipart Parsing | 文件上传 multipart 解析 |

**新增文件**：`src/runtime/http_server.py` (~1481 lines)
