# Nexa v2.1 Agent Properties (新增)

> v2.1.0 为 Agent 新增 4 个编译器和运行时原生支持的属性。

## 属性速查

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `stream` | `true`/`false` | `false` | 启用流式输出，逐 token 实时返回 |
| `output_format` | `"json"`/`""` | `""` | 强制 JSON 结构化输出 |
| `output_schema` | `{...}` | — | JSON Schema 约束，编译器自动生成 Pydantic 模型 |
| `max_tool_calls` | `int` | `10` | 单次请求最大工具调用轮次 |
| `tool_call_strategy` | `"auto"`/`"required"`/`"none"` | `"auto"` | 工具调用策略控制 |

## 使用示例

```nexa
agent Coder {
    prompt: "You are a coding assistant.",
    model: "gpt-4o-mini",
    stream: true,
    max_tool_calls: 5,
    tool_call_strategy: "auto"
}
```

## 编译验证

```bash
nexa build app.nx --harness=warn
```

## 对应 DAG 并行 (已有语法，非新增)

```nexa
// > 串行管道
result = input |> Researcher |> Writer;

// |>> 并行分叉 (v1.0+)
results = input |>> [Agent1, Agent2, Agent3];
## v2.2.1: Context-as-Structure

Agent 声明体内可使用 `context { }` 子块声明上下文行为：

```nexa
agent Researcher {
    model: "minimax-m2.5"
    context {
        source: upstream
        sink: downstream
        output_schema: ResearchOutput
        max_history_turns: 20
        inherit: [messages, artifacts]
    }
    prompt: "..."
}
```

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `source` | string | `upstream` | context 来源 |
| `sink` | string | `downstream` | context 去向 |
| `input_schema` | Protocol | - | 输入类型契约 |
| `output_schema` | Protocol | - | 输出类型契约 |
| `max_history_turns` | int | - | 最大对话轮数 |
| `inherit` | list | `[]` | 继承字段 |

C-004 规则在编译期检查 pipeline 中 A.output_schema 与 B.input_schema 的兼容性。

## v2.3.1: Terminal UI (std.ui)

12 个 UI 函数可用：

```nexa
flow main {
    std.ui.banner("My App");
    std.ui.markdown("# Hello");
    std.ui.thinking("Analyzing", 1.0);
    std.ui.agent_reply("Agent", "Reply");
}
```

| 函数 | 说明 |
|------|------|
| `std.ui.banner(title, subtitle)` | 启动 banner |
| `std.ui.markdown(text)` | Markdown 渲染 |
| `std.ui.code(code, lang)` | 语法高亮代码 |
| `std.ui.panel(text, title, style)` | 样式化面板 |
| `std.ui.thinking(msg, duration)` | spinner 动画 |
| `std.ui.success(msg)` | ✅ 成功 |
| `std.ui.error(msg)` | ❌ 错误 |
| `std.ui.warning(msg)` | ⚠️ 警告 |
| `std.ui.info(msg)` | ℹ️ 信息 |
| `std.ui.input(prompt)` | CJK 兼容输入 |
| `std.ui.agent_reply(name, reply)` | Agent 回复面板 |
| `std.ui.tool_call(name, args)` | 工具调用通知 |
